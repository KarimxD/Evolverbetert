initialize :: [Flag] -> IO (IORef World, Handle) #

Uses a list of flags to initialize a IORef to a world and a handle for output. Generates an initial agent, populates the world wit it Sets the PRNG

main :: IO () #

Maybe initializes graphical display dependent on display in Parameters calls initialize starts mainLoop

mainLoop :: IORef World -> Handle -> Time -> IO () #

Recursive function that reads the IORef of the world, changes it with newWorld and writes to world back. Also writes output if outputTime Might display graphical output using mainLoopEvent from MyGraphics

chEnv :: Env -> Rand Env #
